# Git和SVN提交备注关键字

以下关键字为常用关键字：

| 关键字        | 含义                | 示例                           |
| ---------- | ----------------- | ---------------------------- |
| `feat`     | 新增功能              | `feat(user): 添加用户注册接口`       |
| `add`      | 新增配置/文档文件         | `add: 新增配置文件`                |
| `create`   | 新增模板/初始化文件        | `create: 新增模板文件`             |
| `fix`      | 修复Bug             | `fix(login): 解决登录超时问题`       |
| `docs`     | 文档更新              | `docs(README): 更新安装步骤`       |
| `style`    | 代码样式调整（不涉及逻辑）     | `style: 统一缩进为4空格`            |
| `refactor` | 代码重构（非功能新增/Bug修复） | `refactor(database): 优化查询逻辑` |
| `perf`     | 性能优化              | `perf(image): 压缩图片加载时间`      |
| `test`     | 测试相关              | `test(auth): 添加登录单元测试`       |
| `chore`    | 日常维护（依赖/工具等）      | `chore(deps): 升级依赖版本`        |
| `ci`       | 持续集成配置            | `ci: 配置Action流水线`            |
| `build`    | 构建系统或外部依赖变更       | `build: 更新Webpack配置`         |
| `revert`   | 回退提交              | `revert: 撤销某次错误提交`           |
| `remove`   | 删除文件              | `remove: 删除未使用的配置`           |
